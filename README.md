# 灵签占卜

一个可在微信中通过链接或二维码打开的抽签小应用。扫码进入页面即可抽签查看结果。

## 功能

- 移动端适配，适合微信内置浏览器打开
- 点击「抽一签」随机得到一支签
- 显示签的吉凶等级、签诗与解签说明
- 支持「再抽一签」继续玩

## 本地运行

用本地服务器打开（避免 file 协议下部分功能受限）：

```bash
# 若已安装 Python 3
python3 -m http.server 8080

# 或使用 npx
npx serve .
```

浏览器访问：`http://localhost:8080`

## 部署与分享

1. **部署到任意静态托管**  
   把当前目录下的 `index.html`、`style.css`、`data.js`、`app.js` 上传到：
   - 腾讯云静态网站 / 阿里云 OSS 静态托管
   - GitHub Pages、Vercel、Netlify 等
   - 或公司/个人服务器的 Nginx 静态目录

2. **获得访问链接**  
   部署后得到访问地址，例如：`https://your-domain.com/` 或 `https://xxx.github.io/fortune/`

3. **在微信中分享**
   - **链接**：把上述链接发到微信群或好友，对方点击即可在微信内打开并抽签。
   - **二维码**：用任意「链接转二维码」工具（如草料二维码、联图等）生成二维码，微信扫一扫即可打开同一页面。

## 文件说明

| 文件        | 说明           |
|-------------|----------------|
| `index.html` | 页面结构       |
| `style.css`  | 样式与动效     |
| `data.js`    | 签文数据       |
| `app.js`     | 抽签与展示逻辑 |

签文内容可在 `data.js` 中增删改，格式保持 `label / title / poem / desc` 即可。
